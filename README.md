# Development Mode
## Run the following commands to compiling Elm
```cd frontend```

```elm make src/Main.elm --output ../elm.js```

Open index.html file in your browser

## Run the following commands to execute the Haskell API

```cd backend```

```stack build```

```stack exec -- tasks-api```
